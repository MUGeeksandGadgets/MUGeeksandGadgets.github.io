Projects Underway

M.A.T.T (Mechanical Autonomous Treaded Tank)

M.A.T.T is a self driving robot built on an Arduino platform. M.A.T.T is constructed out of aluminium, rubber, and reinforced cardboard. He is capable of driving around a room and avoiding obstacles without any human intervention.

Project Lead: Patrick Shinn

Linux From Scratch (LFS)

Starting in the Fall semester of 2016, the geeks will begin working on a simple LFS system that will probaly start out as 32bit, but may progress to 64bit eventually given enough free time.

Free Edit

Free Edit is an open source text editor that was forked from Patrick Shinn's Text-Edit. It is written in Java and utilizes Swing for its GUI. Hopefully this can be converted to a JavaFX project in the future, but for now, Swing will do.

Project Lead: Patrick Shinn
