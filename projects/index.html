Projects Underway

WEEKS ON FOOT

WEEKS ON FOOT is a platforming game. It's a reimplementation of the run-and-jump mechanics of Nintendo's Super Mario Bros using libSDL, and it also boasts stability and frugality by moving difficult calculations to Compile time. It parses no INIs, renders no trueType fonts; it just compiles and plays.

Project Lead: Charlie Murphy
Source Code

M.A.T.T (Mechanical Autonomous Treaded Tank)

M.A.T.T is a self driving robot built on an Arduino platform. M.A.T.T is constructed out of aluminium, rubber, and reinforced cardboard. He is capable of driving around a room and avoiding obstacles without any human intervention.

Project Lead: Patrick Shinn
Source Code | Images

5iv

5iv is a small virtual machine for running code of the RV32I subset of the RISC-V Instruction Set Architecture. The virtual machine is connected to a graphical window which can be controlled by SCALL instructions according to a simple ABI. 5iv aspires to be a virtualized game platform similar to the many game console emulators in existence, only with an open instruction set rather than a proprietary one.

Project Lead: Christopher Murphy
